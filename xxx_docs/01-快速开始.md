# 快速开始

## 环境要求

| 依赖 | 版本要求 | 用途 |
|------|----------|------|
| Node.js | >= 20.19.0 | 运行时环境 |
| npm | >= 10 | 包管理 |
| Python | >= 3.10 | Trellis 脚本 |
| uv | 最新版 | Python 包管理 |
| OpenSpec CLI | 全局安装 | 变更管理 |

## 安装步骤

### 1. 安装 OpenSpec CLI（全局）

```bash
npm install -g @fission-ai/openspec@latest
```

### 2. 安装项目依赖

```bash
# Node.js 依赖
npm install

# Python 依赖（使用 uv）
uv sync
```

### 3. 验证环境

```bash
# 快速验证（约 10 秒）
npm run verify:fast

# 完整验证（约 30 秒）
npm run verify

# 健康检查
npm run workflow:doctor
```

### 4. 生成代码库地图

```bash
npm run map:codebase
# 输出: xxx_docs/generated/codebase-map.md
```

## 首次运行检查清单

```bash
# 1. 验证基础功能
npm run verify:fast

# 2. 运行策略门禁
npm run workflow:policy

# 3. 采集指标
npm run metrics:collect

# 4. 检查指标门禁
npm run workflow:gate

# 5. 完整验证
npm run verify
```
