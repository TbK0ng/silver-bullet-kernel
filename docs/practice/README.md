## 阅读约定

- 文中 `.metrics/*` 代表“执行命令后生成”的运行产物；未执行前不存在属于正常情况。
- 文中 `<change>`、`<owner>`、`<target-repo-root>` 为占位符，需替换为你的实际值。

## 选择建议

- 首次接入：从 `01` 开始。
- 存量接管：优先 `03 + 05 + 07`。
- 只看提示词：直接 `06`。
- 只看平台差异/Beta：直接 `08`。
- 只看代码架构与执行链路：直接 `09`。
